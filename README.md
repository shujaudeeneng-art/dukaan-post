# 🏪 Dukaan Post — Free Social Media Post Generator for Shopkeepers

Dukaan Post ek free web-based tool hai jo Pakistan aur South Asia ke chhote dukaandaron aur online business owners ke liye banaya gaya hai. Is tool ki madad se shopkeepers sirf 10 seconds mein apni shop ke liye professional social media captions aur posts generate kar sakte hain — wo bhi Urdu (Roman & Urdu Script) aur English dono mein.

## ✨ Features

- **100% Free:** Koi hosting cost nahi, koi monthly subscription nahi.
- **No API Cost:** Yeh tool pure client-side JavaScript par chalta hai, isliye isme koi paid AI/OpenAI API key nahi chahiye.
- **Multiple Categories:** Kapron ki dukaan (Clothing), Mobile shop, Restaurant, Bakery, Cosmetics, Electronics, Grocery, Shoes, aur Jewelry ke liye special templates.
- **Post Types:** Discount offers, New Arrivals, Eid Sale, Weekend offers, aur Clearance Sale ke liye tayyar captions.
- **Built-in SEO:** Google Search Console, Sitemaps, aur Open Graph tags ke sath pehle se optimized hai taake site Google par rank kar sake.

## 🚀 Technical Architecture

- **Frontend:** HTML5, CSS3 (Modern Responsive Design)
- **Logic:** Vanilla JavaScript (Template-based generation)
- **Hosting:** GitHub Pages (Lifetime Free)

## 📁 Files Structure

- `index.html` — Main landing page aur tool interface.
- `script.js` — Post generation logic aur language switcher.
- `robots.txt` & `sitemap.xml` — Google Search Engine Optimization (SEO) ke liye.
- `404.html` — Custom error page.
